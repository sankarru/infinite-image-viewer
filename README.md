# Infinite Image Viewer

A beautiful infinite scroll image viewer built with Bun.js.

## Features

- Infinite scrolling with lazy loading
- Responsive grid layout
- Hover effects and animations
- Beautiful gradient background
- Powered by Lorem Picsum API

## How to Run

### Install Bun (if not already installed)
```bash
curl -fsSL https://bun.sh/install | bash
```

### Install dependencies and run
```bash
bun install
bun run dev
```

Then open http://localhost:3000 in your browser.

## Deployment

This project can be deployed to GitHub Pages, Vercel, or any static hosting service.

## Tech Stack

- Bun.js - JavaScript runtime
- Vanilla JavaScript - No frameworks
- CSS3 - Modern styling
- Lorem Picsum - Random image API
